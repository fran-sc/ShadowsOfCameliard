# Enemigos

La demo incluye dos variantes principales:

| Enemigo | Descripción |
|---|---|
| `Orc` | Enemigo de aspecto orco. |
| `Skeleton` | Variante visual de esqueleto. |

Ambos comparten estructura técnica, IA y máquina de animaciones equivalente. La diferencia principal es visual y en sonidos de muerte.
