# Escenas

La demo se organiza en cuatro escenas principales.

| Escena | Tipo | Descripción |
|---|---|---|
| [`Countryside`](countryside.md) | Jugable | Zona exterior principal. Contiene managers, UI, Lancelot, recursos, NPC, enemigos, portales y tilemaps. |
| [`CountryCave`](countrycave.md) | Jugable | Zona subterránea más cerrada, con enemigos, agua, antorchas y portales. |
| [`CodexIntro`](codexintro.md) | Narrativa | Códice inicial con páginas de introducción. |
| [`CodexEnd`](codexend.md) | Narrativa | Códice final con páginas adicionales del capítulo jugado. |
